multiblock
==========

Structured learning package for Python 2.7.

The python packages and modules should be compliant with scikit-learn, in a way
such that stable and generic code could be pushed to the scikit-learn
repository.

Dependencies
------------
* Python 2.7.x
* NumPy >= 1.6.1
* Scipy >= 0.9.0
* Scikit-learn >= 0.13
