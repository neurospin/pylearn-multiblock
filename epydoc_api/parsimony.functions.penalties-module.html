Module penalties

source code

The :mod:`parsimony.functions.penalties` module contains the penalties used to constrain the loss functions. These represent mathematical functions and should thus have properties used by the corresponding algorithms. These properties are defined in :mod:`parsimony.functions.interfaces`.

Penalties should be stateless. Penalties may be shared and copied and should therefore not hold anything that cannot be recomputed the next time it is called.
