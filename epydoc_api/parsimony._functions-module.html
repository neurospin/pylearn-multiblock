Module _functions

source code

The :mod:`parsimony.functions` module contains several functions used throughout the package. These represent mathematical functions and should thus have properties used by the corresponding algorithms.

Loss functions should be stateless. Loss functions may be shared and copied and should therefore not hold anything that cannot be recomputed the next time it is called.
