 19      """ Generates random data for regression purposes. Builds data with a 
 20      regression model on the form 
 21   
 22          y = X.beta + e. 
 23   
 24      Parameters 
 25      ---------- 
 26      size : A list or a list of lists. The shapes of the block matrices to 
 27              generate. The numbers of rows must be the same. 
 28   
 29      rho : A scalar or a list of the average correlation between off-diagonal 
 30              elements of S. 
 31   
 32      delta : Baseline noise between groups. Only used if the number of groups is 
 33              greater than one and locally_smooth=False. The baseline noise is 
 34              computed as 
 35   
 36                  delta * rho_min, 
 37   
 38              and you must prvide a delta such that 0 <= delta < 1. 
 39   
 40      eps : Maximum entry-wise random noise. This parameter determines the 
 41              distribution of the noise. The noise is approximately normally 
 42              distributed. If locally_smooth=False the mean is 
 43   
 44                  delta * rho_min 
 45   
 46              and the variance is 
 47   
 48                  (eps * (1 - max(rho))) ** 2.0 / 10. 
 49   
 50              If locally_smooth=True, the mean is zero and the variance is 
 51   
 52                  (eps * (1.0 - max(rho)) / (1.0 + max(rho))) ** 2.0 / 10. 
 53   
 54              You can thus control the noise by this parameter, but note that you 
 55              must have 
 56   
 57                  0 <= eps < 1. 
 58   
 59      density : Determines how much of the regression vector is set to zero. If 
 60              density=1.0, the regression vector is dense and if density=0.0 
 61              would mean a zero vector. However, note that you should let 
 62   
 63                  density * p >= 1, 
 64   
 65              where p is the number of columns in size. 
 66   
 67      snr : The signal-to-noise ratio. The dependent variable is computed as 
 68   
 69                  y = X.beta + e 
 70   
 71              and Var(e) = (||X.beta||² / (n - 1)) / snr. 
 72   
 73      locally_smooth : If True, uses ToeplitzCorrelation (with "local 
 74              smoothing"); if False, uses ConstantCorrelation. 
 75   
 76      Returns 
 77      ------- 
 78      X : The matrix of independent variables. 
 79   
 80      y : The dependent variable. 
 81   
 82      beta : The regression vector. 
 83   
 84      e : The noise/residual vector. 
 85      """ 
 86      if not isinstance(rho, (list, tuple)): 
 87          size = [size] 
 88          rho = [rho] 
 89   
 90      K = len(rho) 
 91   
 92      p = [0] * K 
 93      n = None 
 94      for k in xrange(K): 
 95          if n != None and size[k][0] != n: 
 96              raise ValueError("The groups must have the same number of samples") 
 97          n = size[k][0] 
 98          pk = size[k][1] 
 99          p[k] = pk 
100   
101      if eps == None: 
102          eps = np.sqrt(10.0 / float(n))  # Set variance to 1 / n. 
103   
104      if locally_smooth: 
105          S = correlation_matrices.ToeplitzCorrelation(p, rho, eps) 
106      else: 
107          S = correlation_matrices.ConstantCorrelation(p, rho, delta, eps) 
108   
109      p = sum(p) 
110   
111      # Create X matrix using the generated correlation matrix 
112      mean = np.zeros(p) 
113      X = np.random.multivariate_normal(mean, S, n) 
114   
115      # Apply sparsity 
116      beta = (np.random.rand(p, 1) - 0.5) * 2.0 
117      ind = range(p) 
118      random.shuffle(ind) 
119      ind = ind[:int(round(len(ind) * (1.0 - density)))] 
120      beta[ind] = 0 
121   
122      # Compute pure y 
123      y = np.dot(X, beta) 
124   
125      # Add noise from N(0, (1/snr)*||Xb||² / (n-1)) 
126      var = (np.sum(y ** 2.0) / float(n - 1)) / float(snr) 
127      e = np.random.randn(n, 1) 
128      e *= np.sqrt(var) 
129      y += e 
130   
131      return X, y, beta, e
