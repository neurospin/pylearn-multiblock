 70      Details 
 71      ------- 
 72      The general procedure is: 
 73          1) For each pixel i, Generate independant variables Xi ~ N(0, 1) 
 74          2) Add object level structure corresponding to the five dots: 
 75             - Sample five latent variables ~ N(0, 1): l1, l12, l3, l4, l45. 
 76             - Pixel i of dots X1, X2, X3, X4, X5 are sampled as: 
 77               X1i = l1 + l12 + Xi 
 78               X2i = l12 + Xi 
 79               X3i = 2 * l3 + Xi 
 80               X4i = l4 + l45 + Xi 
 81               X5i = l45 + Xi 
 82               Note that: 
 83               Pixels of dot X1 share a common variance that stem from l1 + l12. 
 84               Pixels of dot X2 share a common variance that stem from l12. 
 85               Pixels of dot X1 and pixel of dot X2 share a common variance that 
 86               stem from l12. 
 87               etc. 
 88          4) Spatial Smoothing. 
 89          5) Model: y = X beta + noise 
 90          - Betas are null outside dots, and 1 or -1 depending on the dot: 
 91              X1: 1, X2: -1, X3: 1, X4: 1, X5: -1. 
 92          - Sample noise ~ N(0, 1) 
 93          - Compute X beta then scale beta such that: r_squared(y, X beta) = r2 
 94          Return X, y, beta 
 95
